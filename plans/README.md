## Plan Lifecycle

### Active Plans
- Plans for ongoing or future work
- Regularly reviewed and updated
- Located in `plans/`

### Archived Plans
- Completed plans
- Superseded plans
- Historical context
- Located in `docs/archive/historical-plans/`

## Creating New Plans

When creating a new plan:

1. **Use Clear Naming**: `COMPONENT_FEATURE_PLAN.md`
2. **Include Metadata**:
   - Status (Active/In Progress/Blocked)
   - Purpose
   - Owner
   - Timeline
3. **Structure**:
   - Overview
   - Goals
   - Implementation phases
   - Success criteria
   - Timeline
4. **Update This README**: Add entry to Current Plans section

## Archiving Plans

When a plan is completed or superseded:

1. Move to `docs/archive/historical-plans/`
2. Update `docs/archive/historical-plans/README.md`
3. Remove from this README
4. Update any cross-references
