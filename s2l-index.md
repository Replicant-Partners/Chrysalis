# S2L Prompt Variants

Use these depending on the agent/runtime:
- `s2l.md` — LangChain/LangGraph + LangSmith, open-source biased, graph design.
- `s2l-vscode-agents.md` — Cline / Roo Code / KiloCode (VS Code toolcalling + MCP).
- `s2l-openai-agents.md` — OpenAI Agents/Assistants (function calling/retrieval/code interpreter).
- `s2l-other-agents.md` — CrewAI (multi-agent), ElizaOS (persona + tools), generic multi-agent.
- `s2l-compound.md` — Master prompt: asks the user to pick a target model and then applies the corresponding template.
